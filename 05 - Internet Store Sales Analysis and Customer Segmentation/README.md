# Исследование продаж и сегментация покупателей интернет-магазина

## Задачи проекта:
На основе данных о продажах интернет-магазина провести сегментацию пользователей, выявить причины снижения выручки.

## Стек:
 - Python
 - Pandas
 - NumPy
 - Matplotlib
 - Seaborn
 - Plotly
